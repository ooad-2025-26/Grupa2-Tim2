## The Last Key

## O projektu
The Last Key je sistem za upravljanje i rezervaciju Escape Room usluga.  
Omogućava pregled dostupnih soba, rezervaciju termina, online plaćanje, ostavljanje recenzija, komunikaciju sa podrškom i primanje obavijesti putem e-maila.

Sistem podržava različite korisničke uloge (Gost, Korisnik, Zaposlenik i Administrator) te omogućava upravljanje sobama, terminima, rezervacijama, recenzijama, obavijestima i popustima.
